## 安装

Maven 依赖：

```xml
<dependency>
  <groupId>com.oceanbase</groupId>
  <artifactId>obvector4j</artifactId>
  <version>1.0.7</version>
</dependency>
```

源码安装：

```bash
git clone https://github.com/oceanbase/obvector4j.git
cd obvector4j
make build
```

## 连接数据库

```java
import com.oceanbase.obvector4j.ObVecClient;

String uri = System.getenv("OCEANBASE_URI");       // jdbc:oceanbase://<host>:<port>/<database>
String user = System.getenv("OCEANBASE_USER");
String password = System.getenv("OCEANBASE_PASSWORD");

ObVecClient client = new ObVecClient(uri, user, password);
```

## 建表与写入

```java
import com.oceanbase.obvector4j.schema.*;

ObCollectionSchema schema = new ObCollectionSchema();
ObFieldSchema id = new ObFieldSchema("c1", DataType.INT32);
id.IsPrimary(true).IsAutoInc(true);
ObFieldSchema vec = new ObFieldSchema("c2", DataType.FLOAT_VECTOR);
vec.Dim(3);
schema.addField(id);
schema.addField(vec);

IndexParams indexes = new IndexParams();
indexes.addIndex(new IndexParam("vidx1", "c2"));
schema.setIndexParams(indexes);

client.createCollection("my_table", schema);
```

```java
import com.oceanbase.obvector4j.model.*;

ArrayList<Sqlizable[]> rows = new ArrayList<>();
rows.add(new Sqlizable[] { new SqlVector(new float[] {1f, 2f, 3f}) });
client.insert("my_table", new String[] {"c2"}, rows);
```

## 向量检索

```java
ArrayList<HashMap<String, Sqlizable>> results = client.query(
    "my_table", "c2", "l2",
    new float[] {0f, 0f, 0f}, 10,
    new String[] {"c1", "c2"},
    new DataType[] { DataType.INT32, DataType.FLOAT_VECTOR },
    null);
```

## 混合搜索（常用）

```java
import com.oceanbase.obvector4j.filter.*;

// 标量 + 向量
Filter filter = FilterBuilder.key("category_id").isEqualTo(1);
client.scalarVectorSearch()
    .table("products")
    .queryVector(queryVector)
    .filter(filter)
    .topk(10)
    .outputFields("id", "name")
    .search();

// 全文 + 向量
client.textVectorSearch()
    .table("documents")
    .queryVector(queryVector)
    .textFields("title", "content")
    .textQuery("关键词")
    .topk(10)
    .search();
```

详细说明见 [混合搜索](混合搜索.md) 与 [Filter过滤](Filter过滤.md)。

## 自定义 SQL 与 DSL

### 自定义 SQL

```java
client.executeSql("DELETE FROM my_table WHERE id = 1");
ArrayList<HashMap<String, Sqlizable>> rows = client.querySql("SELECT c1, c2 FROM my_table LIMIT 5");
```

### 自定义 HYBRID_SEARCH DSL（4.6.0+，低于 4.6.0 报错）

```java
import com.oceanbase.obvector4j.hybrid.core.dsl.HybridDsl;

client.hybridSearch()
    .customSearch()
    .table("documents")
    .query(HybridDsl.match("content", "关键词"))
    .knn(HybridDsl.knn("c2", queryVector, 10))
    .rank(HybridDsl.rrf(10, 60))
    .size(10)
    .outputFields("c1", "c2")
    .search();
```

特殊语法仍可用 `queryDsl()` / `knnDsl()` 传入原始 JSON 兜底。详见 **[HYBRID_SEARCH DSL 语法](dsl语法.md)**（完整 JSON 规则与示例）。
